# C_Banking_with_RSA_encryption
Simple banking system program written in C with RSA encryption

# Dependencies  
gtk for GUI program.  
  
# How to run  
FOR CLI : gcc -std=c11 cli.c  
LINUX : ./a.out  
WINDOWS : a  
  
FOR GUI : gcc -std=c11 gui.c -o gui `pkg-config --cflags --libs gtk+-3.0`  
./gui
